# 00 — Bootstrap Frontend

## Objetivo
Criar a base do frontend antes dos módulos de negócio.

## Checklist

1. criar repositório Git
2. iniciar Vite com React + TypeScript
3. instalar Ant Design
4. configurar React Router DOM
5. configurar Axios
6. configurar React Hook Form + Zod
7. configurar Zustand
8. configurar Vitest + Testing Library
9. configurar coverage
10. configurar Sonar
11. criar shell administrativo
12. criar layout base
13. criar rota do dashboard vazia
14. criar cliente HTTP base
15. criar estados de loading/erro/empty
16. garantir build e testes verdes

## Resultado esperado
Frontend sobe localmente com shell administrativo e testes base passando.
